# Golang-gRPC


## ⚙️ Requirements

- Go 1.18+
- Git

---

## 🚀 Getting Started


### 1. Clone the repo

```bash
git clone https://github.com/rzmsv/golang-grpc.git
```
### 2. Run application
1. For install dependencies run this command :
    ```bash
    go mod tidy
    ```
2. For run server :
    ```bash
    go run /server/main.go
    ```
3. For run client :
    ```bash
    go run /client/main.go
    ```

** Notice: App run on port 8080 statically **
